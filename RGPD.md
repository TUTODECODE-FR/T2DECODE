# Politique de Confidentialité et RGPD (T2DECODE)

## 1. Préambule
L'association **TUTODECODE** (SIREN : 102 763 133) s'engage à protéger la vie privée de ses utilisateurs. 
Le projet **T2DECODE** est conçu selon le principe de **Privacy by Design** (protection de la vie privée dès la conception).

## 2. Collecte de données
T2DECODE est une application **Offline-First**. 
- **Aucune donnée personnelle** n'est collectée, traitée ou transmise à des serveurs tiers par l'application.
- **Aucun outil de suivi** (tracker) ou d'analyse d'audience (analytics) n'est intégré au code de l'application.
- Toutes vos données (progression dans les cours, préférences de l'utilisateur, historique de navigation, requêtes à l'IA locale via Ollama) restent stockées **localement** sur votre propre appareil.

## 3. Fonctionnalités Réseau
Certaines fonctionnalités optionnelles (comme **Ghost Link**) utilisent le réseau local (LAN) ou des connexions directes :
- Ces connexions sont limitées au réseau local ou pair-à-pair.
- Les données transmises via ces fonctionnalités sont chiffrées et ne transitent par aucun serveur de l'association.
- L'association TUTODECODE n'a **jamais accès** à ces flux réseau.

## 4. Intelligence Artificielle Locale (Ghost AI)
Le module **Ghost AI** fonctionne exclusivement avec des modèles locaux via [Ollama](https://ollama.ai/).
- Vos prompts, questions et données sensibles ne quittent jamais votre machine.
- Aucune API cloud n'est utilisée (Zéro Cloud Policy).

## 5. Vos Droits (RGPD)
Puisque nous ne collectons ni ne stockons aucune donnée personnelle sur des serveurs distants, l'exercice des droits classiques du RGPD (droit d'accès, de rectification, d'effacement) s'effectue directement par l'utilisateur :
- En supprimant l'application de votre appareil.
- En effaçant les données locales (Clear Data) depuis les paramètres de votre système d'exploitation.

## 6. Mises à jour & Télémétrie
Si l'application télécharge des mises à jour de modules ou de "cheat sheets", elle interroge le dépôt GitHub public de TUTODECODE de manière standard. GitHub peut collecter des adresses IP lors du téléchargement des assets, conformément à sa propre politique de confidentialité. Il n'y a **aucune télémétrie** intégrée par TUTODECODE.
