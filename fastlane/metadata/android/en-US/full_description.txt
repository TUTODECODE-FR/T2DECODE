TUTODECODE is an offline-first technical learning platform and cybersecurity toolbox.

Built for constrained or air-gapped environments:
- no cloud dependency
- no analytics or tracking
- local AI with Ollama
- local-first educational content

Included features:
- local courses rendered from Markdown and JSON
- offline tools for networking, system and cybersecurity tasks
- interactive labs for networking, Linux, security, cloud and cryptography basics
- LAN-only peer communication with GhostLink

Network behavior is limited to local use cases:
- localhost for Ollama local models
- local network for GhostLink peer discovery and messaging
- no third-party telemetry endpoints
