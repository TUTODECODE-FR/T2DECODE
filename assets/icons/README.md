# Structure des Ressources (Assets)

Ce répertoire est organisé pour une gestion professionnelle des ressources :

- `logo.png` : Le logo principal de la marque (1024x1024 recommandé).
- `icons/` : Contient les icônes de lancement spécifiques aux plateformes.
  - Exécutez `flutter pub run flutter_launcher_icons` pour les régénérer à partir de `logo.png`.
- `splash/` : Contient les images de l'écran de démarrage (splash screen).
  - Exécutez `flutter pub run flutter_native_splash:create` pour les mettre à jour.
- `fonts/` : Typographie personnalisée.
