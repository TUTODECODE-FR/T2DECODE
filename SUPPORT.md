# Support T2DECODE

Merci d'utiliser T2DECODE ! En tant que projet à but non lucratif et open-source, nous mettons tout en œuvre pour fournir un support de qualité à notre communauté. 

Voici comment obtenir de l'aide ou signaler un problème :

## 1. Documentation et Wiki
Avant d'ouvrir un ticket, merci de consulter notre documentation intégrée au projet :
- Lisez le [README.md](README.md) pour les instructions d'installation et de compilation.
- Consultez notre Wiki sur GitHub (s'il est activé).
- Explorez les commentaires du code et les documents dans le dossier `/docs`.

## 2. Signaler un Bug (Issue)
Si vous avez trouvé un bug ou si l'application plante :
1. Allez dans l'onglet **Issues** de notre dépôt GitHub.
2. Cherchez si le problème n'a pas déjà été signalé.
3. Si ce n'est pas le cas, cliquez sur **New Issue**.
4. Décrivez le problème avec le plus de détails possibles (votre système d'exploitation, la version de l'application, et les étapes pour reproduire le bug).

## 3. Demande de Fonctionnalité
Si vous avez une idée géniale pour améliorer l'application ou ajouter un nouveau cours/outil :
1. Ouvrez une **Issue** avec le tag `enhancement`.
2. Expliquez clairement le besoin et comment cette fonctionnalité pourrait aider les utilisateurs.

## 4. Vulnérabilité de Sécurité
Si vous découvrez une faille de sécurité, **merci de ne pas ouvrir de ticket public**. 
Veuillez vous référer à notre politique de sécurité dans [SECURITY.md](SECURITY.md) et nous contacter directement en privé via nos canaux sécurisés.

## 5. Communauté et Contact
Pour des questions plus générales ou pour discuter du projet :
- Contactez l'association à **contact@tutodecode.org**
- Consultez notre site officiel [https://tutodecode.org](https://tutodecode.org)

Merci de contribuer à rendre T2DECODE meilleur pour tous !
