# Build & Tests

## Prérequis
- Flutter SDK
- Dart SDK
- Make

## Installation
```bash
git clone https://github.com/TUTODECODE-FR/T2DECODE.git
cd T2DECODE
make get
```

## Tests
```bash
make test
```

## Builds
```bash
make build-android
make build-android-fdroid
make build-macos
make build-linux
make build-dmg
```
