# Politique de confidentialité — T2DECODE

Dernière mise à jour : 17 avril 2026

## 1. Responsable
TUTODECODE (Association TUTODECODE)

## 2. Principe général
T2DECODE est conçu avec une approche **offline-first** et **zéro tracking**.

- Aucune collecte de données personnelles à des fins publicitaires.
- Aucune vente de données.
- Aucune télémétrie vers des services tiers.

## 3. Données collectées
L'application **ne collecte pas** de données personnelles sur des serveurs distants.

Les données fonctionnelles (préférences, progression, configuration locale) sont stockées localement sur l'appareil de l'utilisateur.

Exemples de données locales :
- préférences (thème, options, tailles de police)
- états de progression
- configuration de fonctionnalités locales (ex : URL Ollama si activée)

## 4. Réseau et connexions
L'application peut effectuer des communications réseau limitées dans les cas suivants :

- **Ollama local** : connexion à `localhost` ou à une adresse locale configurée par l'utilisateur.
- **Ghost Link** : communication sur réseau local (LAN) entre appareils.

Aucune API cloud tierce n'est utilisée pour le fonctionnement principal.
En dehors d’Ollama (local) et Ghost Link (LAN), T2DECODE n’effectue pas de connexions vers des services externes.

## 5. Permissions iOS
Selon les fonctionnalités utilisées, l'application peut demander :

- Accès à la photothèque (sélection/export d'images).
- Accès au réseau local (découverte/connexion entre appareils sur LAN).

Ces permissions sont utilisées uniquement pour les fonctions demandées par l'utilisateur.

## 6. Sécurité
Nous mettons en œuvre des mécanismes de vérification d'intégrité des ressources de l'application et des protections contre la modification non autorisée.

## 7. Conservation
Les données locales restent sur l'appareil de l'utilisateur jusqu'à suppression manuelle de l'app ou des contenus concernés.

## 8. Vos droits
Vous pouvez :

- Refuser les permissions système depuis iOS.
- Supprimer les données locales en supprimant l'application.
- Nous contacter pour toute question liée à la confidentialité.
