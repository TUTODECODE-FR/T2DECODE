# Support — T2DECODE

Merci d'utiliser T2DECODE.

## Statut App Store (iOS)
La version iOS est **en cours de vérification Apple (App Store)** et arrivera prochainement.

## Besoin d'aide
- Ouvrir un ticket : https://github.com/TUTODECODE-FR/T2DECODE/issues
- Documentation : https://github.com/TUTODECODE-FR/T2DECODE/tree/main/docs

## Informations à fournir pour un bug
1. Plateforme (iOS, Android, macOS, Windows, Linux)
2. Version de l'app (ex: 1.0.1)
3. Étapes pour reproduire le problème
4. Résultat attendu vs résultat obtenu
5. Capture d'écran si possible

## FAQ rapide

- **L’IA ne répond pas / Ollama non détecté** : c’est normal si Ollama n’est pas lancé. L’IA locale est optionnelle.
- **Je suis en environnement air‑gapped** : utilisez le mode hors-ligne / mode zéro réseau, et installez les contenus localement.
