## Description
TUTODECODE est une application d'apprentissage technique conçue pour fonctionner en local, avec une approche offline-first.

Fonctionnalités principales :
- Cours et contenus pédagogiques lisibles directement dans l'application
- Outils techniques intégrés pour la pratique
- Laboratoires interactifs
- IA locale via Ollama (sans dépendance cloud obligatoire)
- Fonctionnement orienté souveraineté numérique et zéro tracking

L'application est pensée pour aider à apprendre, pratiquer et progresser de façon structurée, avec une interface claire et des ressources accessibles.
